# RRT and RRT* Path Planning with Real-time Visualization

Working implementations of RRT and RRT* algorithms with real-time visualization and video recording.

## Quick Start

```bash
pip install -r requirements.txt
python3 rrt_working.py      # Run RRT
python3 rrt_star_working.py # Run RRT*
```

## Features
- Real-time tree growth visualization
- Colorful maps (purple free space, yellow obstacles)
- Video recording to ~/Videos/
- 5-second pause after path finding
